 Sales & Demand Forecasting using Machine Learning
 Project Overview

This project focuses on forecasting future sales using historical business data.
Accurate sales forecasting helps businesses plan inventory, manage supply chains, and make informed financial decisions.

The model is built using real-world store sales data and applies machine learning techniques to identify patterns and trends over time.

Dataset

Dataset Name: Store Sales – Time Series Forecasting

Source: Kaggle

Due to GitHub file size limitations, the dataset is not uploaded to this repository.

Dataset Link:
https://www.kaggle.com/competitions/store-sales-time-series-forecasting

Tools & Technologies Used

Python

Google Colab / Jupyter Notebook

Pandas & NumPy (Data Processing)

Matplotlib (Data Visualization)

Scikit-learn (Machine Learning)

 Approach

Loaded and explored historical store sales data

Performed data cleaning and preprocessing

Extracted time-based features such as year, month, day, and day of the week

Encoded categorical variables for model compatibility

Trained a Random Forest Regressor for sales prediction

Evaluated the model using Mean Absolute Error (MAE)

Visualized actual vs predicted sales to understand model performance

 Results & Insights

The model successfully captures sales trends based on historical data and promotional activity

Forecasting results can help businesses:

Reduce overstock and stockouts

Improve demand planning

Optimize promotional strategies


Clear visualizations were created to make the results easy to understand for business users.
