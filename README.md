# 📊 Sales & Demand Forecasting using Machine Learning

## 📌 Project Overview
This project focuses on forecasting future sales using historical business data. Accurate sales forecasting helps businesses plan inventory, manage supply chains, and make informed decisions.

The solution is built using real-world store sales data and applies machine learning techniques to identify patterns and trends over time.

---

## 📂 Dataset
- **Dataset Name:** Store Sales – Time Series Forecasting  
- **Source:** Kaggle  

Due to GitHub file size limitations, the dataset is not included in this repository.

🔗 Dataset Link:  
https://www.kaggle.com/competitions/store-sales-time-series-forecasting

---

## ⚙️ Tools & Technologies Used
- Python  
- Google Colab / Jupyter Notebook  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 🔍 Methodology
1. Loaded and explored historical sales data  
2. Cleaned and preprocessed the dataset  
3. Performed time-based feature engineering  
4. Encoded categorical variables  
5. Trained a Random Forest Regressor for forecasting  
6. Evaluated the model using Mean Absolute Error (MAE)  
7. Visualized actual vs predicted sales  

---

## 📈 Results & Business Insights
- The model captures sales trends based on historical data  
- Forecasting helps businesses:
  - Improve inventory planning  
  - Reduce stock shortages and excess inventory  
  - Support better decision-making
